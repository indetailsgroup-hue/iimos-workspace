- ‚úÖ ‡πÇ‡∏î‡∏¢‡∏¢‡∏±‡∏á‡∏Ñ‡∏á‡πÉ‡∏ä‡πâ Low-poly BoxGeometry (12 Polygons) ‡πÄ‡∏´‡∏°‡∏∑‡∏≠‡∏ô‡πÄ‡∏î‡∏¥‡∏°
- ‚úÖ ‡πÄ‡∏õ‡πá‡∏ô‡∏ß‡∏¥‡∏ò‡∏µ‡∏ó‡∏µ‡πà‡∏õ‡∏£‡∏∞‡∏´‡∏¢‡∏±‡∏î‡∏ó‡∏£‡∏±‡∏û‡∏¢‡∏≤‡∏Å‡∏£‡∏ó‡∏µ‡πà‡∏™‡∏∏‡∏î‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö‡∏Å‡∏≤‡∏£‡∏ó‡∏≥ Instancing ‡∏à‡∏≥‡∏ô‡∏ß‡∏ô‡∏°‡∏´‡∏≤‡∏®‡∏≤‡∏•

```glsl
varying vec3 vPosition;
varying vec3 vNormal;

void main() {
  // ‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì‡∏£‡∏∞‡∏¢‡∏∞‡∏´‡πà‡∏≤‡∏á‡∏à‡∏≤‡∏Å‡∏Ç‡∏≠‡∏ö (Edge Distance)
  vec3 absPos = abs(vPosition);

  // ‡∏ñ‡πâ‡∏≤‡πÉ‡∏Å‡∏•‡πâ‡∏Ç‡∏≠‡∏ö ‡πÉ‡∏´‡πâ‡∏î‡∏±‡∏î‡πÅ‡∏õ‡∏•‡∏á Normal

  // ‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì‡πÅ‡∏™‡∏á‡∏ï‡∏≤‡∏°‡∏õ‡∏Å‡∏ï‡∏¥
  vec3 light = normalize(vec3(1.0, 1.0, 1.0));

  gl_FragColor = vec4(vec3(dProd), 1.0);
}
```
